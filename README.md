# API Locker — Landing Page

> Marketing landing page for [API Locker](https://apilocker.dev) Chrome extension.

**Live:** [apilocker.dev](https://apilocker.dev)

## Stack

- Pure HTML/CSS/JS — zero dependencies
- Hosted on [Vercel](https://vercel.com) (free tier)
- Domain: [Porkbun](https://porkbun.com)
- Waitlist: [Formspree](https://formspree.io)

## Local Development

```bash
open index.html
```

No build step required.

## Deploy

Every push to `main` auto-deploys via Vercel.

## Pages

- `/` — Main landing page
- `/privacy.html` — Privacy Policy
